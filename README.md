# Advanced Calculator (Python)

A command-line based advanced calculator built using Python that supports multiple operations and dynamic user inputs.

Features:
- Perform Addition, Subtraction, Multiplication, Division
- Supports multiple numbers input
- Dynamic user input system
- Error handling (Division by Zero)
- Interactive CLI interface

Tech Stack:
- Python
- Loops
- Lists
- Exception Handling

How to Run:
1. Clone the repository:
git clone https://github.com/databykunal/advanced-calculator-python.git

2. Run the Python file:
python calculator.py

Example:
Choose operation:
1. Addition
2. Subtraction
3. Multiplication
4. Division

Enter numbers → Output result
